## Features

- [x] Live Bus Locations (updated every 15 seconds)
- [x] View All Bus Stops
- [x] Search in All Stops
- [x] Search for location and create public transportation route
- [x] Add NFC card and read balance

## Requirements

- iOS 15.5+
- Xcode 13.4.1

## Installation

#### CocoaPods
You can use [CocoaPods](http://cocoapods.org/) to install `SideMenu, Alamofire, DropDown, Firebase` by adding it to your `Podfile`:

```ruby
  use_frameworks!

  pod 'SideMenu'
  pod 'Alamofire'
  pod 'DropDown'
  pod 'FirebaseAnalytics'
  pod 'FirebaseAuth'
  pod 'FirebaseFirestore'
  pod 'Firebase/Database'
  pod 'FloatingPanel'
```
